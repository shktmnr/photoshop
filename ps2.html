Photoshop: Basic Tools Overview, Selections, and Layers

Course Catalog Day-wise Breakdown

Lesson Overview

This lesson introduces Photoshop's essential tools, focusing on navigation and usage of the Move, Marquee, and Lasso tools. You’ll also learn about selections and how to work with layers, key concepts in Photoshop for both beginners and professionals.

Basic Tools Overview

1. Move Tool

The Move Tool (Shortcut: V) is used to move objects or layers within the canvas. Simply select the Move Tool from the toolbar, click on an object or layer, and drag it to your desired location.

2. Marquee Tool

The Marquee Tool is for making rectangular or elliptical selections. It includes:

  • Rectangular Marquee Tool (M): Draws rectangular or square selections.
  • Elliptical Marquee Tool: Creates circular or oval selections.

3. Lasso Tool

The Lasso Tool lets you create freeform selections. It includes:

  • Lasso Tool (L): Freehand selection of any shape.
  • Polygonal Lasso Tool: Selection with straight edges, useful for geometric shapes.
  • Magnetic Lasso Tool: Automatically snaps to edges for precise selections.

4. Quick Selection and Magic Wand Tools

The Quick Selection Tool and Magic Wand Tool allow for selections based on color and tone:

  • Quick Selection Tool (W): Click and drag to select areas of similar color and texture.
  • Magic Wand Tool: Selects areas with a similar color range.

5. Crop Tool

The Crop Tool (Shortcut: C) lets you trim or resize an image by removing unwanted outer areas.

Working with Selections

1. Making a Selection

Selections are essential in Photoshop, as they allow you to isolate specific areas for editing. Use tools like Marquee, Lasso, or Quick Selection for this. Hold Shift to add to a selection, or Alt (Option) to subtract.

2. Modifying a Selection

Once you’ve made a selection, you can modify it by:

  • Feathering: Softens the edges of a selection.
  • Expand/Contract: Enlarges or shrinks the selection.

3. Saving and Loading Selections

Selections can be saved for future use. Go to Select > Save Selection to save it and Select > Load Selection to reload it later.

Introduction to Layers

1. Understanding Layers

Layers are like transparent sheets stacked on top of each other. They allow you to work on individual parts of an image without affecting others. Layers are managed in the Layers Panel.

2. Creating and Managing Layers

To create a new layer, click on the New Layer button at the bottom of the Layers Panel. You can duplicate, delete, and rename layers as well.

3. Layer Types

  • Image Layers: Contain pixel-based images.
  • Adjustment Layers: Apply adjustments (e.g., brightness, contrast) non-destructively.
  • Text Layers: Contain editable text elements.
  • Shape Layers: Contain vector-based shapes.

4. Working with Layer Masks

Layer masks are used to hide or reveal parts of a layer without permanently erasing content. They are essential for non-destructive editing and are added by clicking the Add Layer Mask icon in the Layers Panel.

5. Layer Blending Modes

Blending modes affect how a layer’s pixels interact with the layers below. Experimenting with blending modes can create different visual effects, such as Overlay, Multiply, and Screen.
