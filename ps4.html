Photoshop: Color Modes, Basic Color Correction, and Layer Masks

Lesson Overview

This lesson covers essential concepts in Photoshop such as Color Modes, Basic Color Correction, and Layer Masks. These tools and concepts are foundational for creating high-quality images and making targeted adjustments to your photos and designs.

Color Modes

1. Introduction to Color Modes

Photoshop offers different color modes that affect how colors are represented in an image. The most commonly used modes are RGB and CMYK. Selecting the correct color mode depends on the final use of the imageâ€”whether for web or print.

2. RGB Color Mode

RGB (Red, Green, Blue) is used for digital displays. It combines red, green, and blue light to create colors on screens. This mode is best for web images and digital artwork.

3. CMYK Color Mode

CMYK (Cyan, Magenta, Yellow, Black) is used for printing. It combines four ink colors to produce a range of colors. This mode is best for images intended for print, as it ensures colors are accurate on paper.

4. Grayscale and Bitmap Modes

Grayscale: Converts images to shades of gray, removing color information. Useful for black-and-white photos.

Bitmap: Converts images to only black and white, with no shades of gray. Useful for creating line art and icons.

Basic Color Correction

1. Introduction to Color Correction

Color correction involves adjusting the colors, contrast, and brightness of an image to enhance its appearance. Photoshop offers multiple tools for achieving this, like Levels, Curves, and the Hue/Saturation adjustment.

2. Using Levels for Color Correction

The Levels adjustment allows you to modify shadows, midtones, and highlights in an image.

  • Go to Image > Adjustments > Levels or use the shortcut Ctrl+L (Cmd+L on Mac).
  • Adjust the sliders under Input Levels to control the shadows, midtones, and highlights.

3. Using Curves for Advanced Color Adjustment

The Curves tool offers greater control for adjusting color and contrast. It allows you to make targeted adjustments at specific points in the tonal range.

  • Go to Image > Adjustments > Curves or use the shortcut Ctrl+M (Cmd+M on Mac).
  • Click on the curve and drag it up or down to adjust brightness or contrast at different points.

4. Adjusting Hue/Saturation

Hue/Saturation adjustments allow you to change the overall color tone and saturation of an image.

  • Go to Image > Adjustments > Hue/Saturation or use the shortcut Ctrl+U (Cmd+U on Mac).
  • Adjust the sliders for Hue, Saturation, and Lightness to get the desired effect.

Layer Masks

1. Introduction to Layer Masks

Layer Masks are essential for non-destructive editing. They allow you to hide or reveal parts of a layer without permanently erasing any pixels, giving you flexibility to refine edits.

2. Creating a Layer Mask

To create a Layer Mask:

  • Select the layer you want to mask in the Layers Panel.
  • Click on the Add Layer Mask button at the bottom of the Layers Panel.
  • A white thumbnail will appear next to the layer, representing the mask.

3. Using Brushes to Refine the Mask

With the Layer Mask selected, you can use the Brush Tool (B) to paint on the mask.

  • Use black to hide parts of the layer and white to reveal them. Gray shades will make the area partially transparent.
  • Adjust brush size and hardness in the options bar to refine edges and details.

4. Refining Masks with Select and Mask

The Select and Mask workspace allows you to fine-tune mask edges, especially for complex areas like hair and fur.

  • With the mask selected, go to Select > Select and Mask.
  • Use tools like the Refine Edge Brush and adjust settings for smoothness and feathering.
