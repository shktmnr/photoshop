Lesson Overview

As we wrap up the week's lessons, we'll recap the essential Photoshop tools we've covered and give you a hands-on project to practice and reinforce your skills. This project will involve a simple image edit, applying basic techniques in selection, color correction, and layer manipulation.

Recap of Essential Tools

1. Selections

This week, you learned about using different selection tools such as the Marquee, Lasso, and Quick Selection tools. These are essential for isolating parts of an image for targeted edits.

  • Marquee Tool: Great for selecting rectangular or elliptical areas.
  • Lasso Tool: Freehand selection, useful for irregular shapes.
  • Quick Selection Tool: Quickly selects areas based on similar colors and tones.

2. Layers and Layer Masks

Understanding layers and masks is crucial for non-destructive editing. Layers allow you to organize different elements of your design separately, and layer masks give you fine control over which parts of each layer are visible.

3. Color Modes and Correction

Choosing the correct color mode (RGB for screens and CMYK for print) ensures accurate color representation. You also learned about basic color correction tools like Levels, Curves, and Hue/Saturation.

4. Healing and Fixing Tools

Tools like the Healing Brush, Patch Tool, and Clone Stamp are essential for retouching photos and removing unwanted spots or blemishes.

Weekend Project: Simple Image Edit

1. Project Objective

For this project, you will perform a basic image edit that includes cleaning up blemishes, adjusting colors, and adding a creative overlay. Choose a photo of your own or use a sample image to get started.

2. Steps to Complete the Project

Step 1: Open and Duplicate the Image

Open your image in Photoshop. Create a duplicate layer by right-clicking on the background layer and selecting Duplicate Layer. This way, you can make edits non-destructively.

Step 2: Apply Color Correction

Adjust the colors and contrast of your image using Levels or Curves.

  • Go to Image > Adjustments > Levels (or use Ctrl+L) and adjust the sliders for the desired effect.
  • Alternatively, use Curves (Ctrl+M) for more targeted adjustments.

Step 3: Clean Up Imperfections

Use the Spot Healing Brush Tool to remove any unwanted blemishes or spots on the image.

  • Select the Spot Healing Brush Tool from the toolbar (press J).
  • Click on any spots or blemishes to let Photoshop automatically blend surrounding pixels.

Step 4: Create a Selection and Add an Overlay

Create a selection of the main subject using the Quick Selection Tool.

  • Select the Quick Selection Tool (press W) and brush over the subject to create a selection.
  • Add a new layer above your image and fill it with a color or gradient.
  • Adjust the blending mode and opacity to create a subtle overlay effect.

3. Submitting Your Project

Save your edited image as a JPEG or PNG. For best quality, save the PSD file as well, keeping your layers intact so you can revisit and refine your project in the future.
